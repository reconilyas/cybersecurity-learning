   # Network Security Fundamentals

# System Security Best Practices

- Keep systems up to date:
  - Firmware
  - Operating systems
  - Drivers

---

## 🔐 Password Security

- Use strong passwords
- Minimum recommended length: **15 characters or more**
- Combine:
  - Letters
  - Numbers
  - Symbols

---

## 🔐 VPN Security Concept

A VPN provides secure communication over a network.

It can also support authentication methods such as:
- Something you know (password)
- Something you have (token/device)
- Something you are (biometrics)

---

## 👤 User Access Control

User access defines what resources a user can access based on their job role.

---

## 🚫 Inactive Accounts

- Inactive accounts should be:
  - Disabled
  - Or deleted

Purpose:
- Reduce security risks
- Prevent unauthorized access

---

# AAA Security Model

## 🔐 Authentication
Verifies the identity of a user.

Example:
- Username + password

---

## 🔑 Authorization
Defines what a user is allowed to do after authentication.

Example:
- Access to specific files or systems

---

## 📊 Accounting
Tracks user activity in the system.

Includes:
- Logs
- Monitoring
- Activity tracking

---

# System Security Tools

## 🔥 Firewall

A firewall controls incoming and outgoing network traffic.

### Functions:
- Filters inbound and outbound traffic
- Allows or blocks connections based on rules

### Types:
- Hardware firewall
- Software firewall

---

## 🛡️ Antivirus Software

- Detects malicious software (malware)
- Removes or quarantines threats
- Must be kept updated for effectiveness

---

# Encryption & Cryptography Basics

## 🔐 Cryptography
Cryptography is the science of securing information using encoding techniques.

Examples:
- Caesar cipher
- Enigma-like systems

---

## 🔒 Encryption

Encryption converts readable data into unreadable format to protect it.

- Plaintext → original data
- Ciphertext → encrypted data

---

## 🔑 Key Concepts

- Public Key → used for encryption
- Private Key → used for decryption

---

## 🏦 Real-world Use Cases

- Banking systems
- Credit card transactions
- Secure online communication

---

## 🧠 Summary

Network security focuses on protecting systems, data, and users using authentication, authorization, encryption, firewalls, and regular system updates. Cryptography ensures data confidentiality and integrity in modern networks.
