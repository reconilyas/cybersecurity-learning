# DHCP & DNS Fundamentals

---

## IP Address Assignment

### Static Addressing

Static addressing is the manual assignment of an IP address to a device.

#### Characteristics
- Assigned manually
- Does not change unless configured manually
- Commonly used for servers, routers, printers, and network devices

---

### Dynamic Addressing

Dynamic addressing automatically assigns an IP address to a device.

#### Characteristics
- Assigned automatically
- Managed by a DHCP server
- Easier to manage in large networks

---

## DHCP (Dynamic Host Configuration Protocol)

DHCP automatically provides network configuration to devices.

It can assign:
- IP address
- Subnet mask
- Default gateway
- DNS server
