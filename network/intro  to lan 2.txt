# Network Connectivity Devices & Cloud Models

---

## 🧩 Network Interface Card (NIC)
A Network Interface Card is a hardware component that connects a computer to a network.

- Also called: Network Adapter  
- Responsible for sending and receiving data over a network  

---

## 🔄 Transceiver
A transceiver is a device that can both transmit and receive signals.

- Often built into network cards  
- Converts signals (example: electrical ↔ optical in fiber networks)  
- Example: GBIC (Gigabit Interface Converter)

---

## 🔀 Switch
A switch connects multiple devices in a network.

- Uses MAC addresses (48-bit hexadecimal)  
- Sends data only to the correct device (not all devices like a hub)  
- Acts as a central connection point in a LAN  

---

## 🌐 Gateway
A gateway connects different networks together.

- Often part of a router  
- Allows communication between different network systems  

---

## 🖥️ Virtualization
Virtualization is the creation of virtual versions of physical systems.

### Examples:
- Virtual Machine (VM)  
- Virtual Server  
- Virtual Switch  
- Virtual Desktop  
- Virtual Router  

---

## ☁️ Cloud Service Models

### NaaS (Network as a Service)
Provides networking services over the cloud.

---

### IaaS (Infrastructure as a Service)
Provides virtual infrastructure like servers and storage.  
Example: Amazon Web Services (AWS)

---

### PaaS (Platform as a Service)
Provides a platform to develop and deploy applications.

---

### SaaS (Software as a Service)
Provides software over the internet.  
Example: Gmail

---

## 🧱 Legacy Networking Devices

- Hub → sends data to all devices  
- Repeater → boosts signal strength  
- Bridge → connects two network segments  

---

## ⚙️ Advanced Networking Devices

### Wireless Access Point (WAP)
Allows wireless devices to connect to a network.

---

### Wireless Controller
Manages multiple wireless access points.

---

### Load Balancer
- Distributes traffic across multiple servers  
- Uses a virtual IP address  
- Ensures availability: if one server fails, traffic is moved to others  

---

## 🧠 Summary
Networking devices and cloud models are essential for understanding how modern networks and cloud systems operate, from basic connectivity to advanced infrastructure.
