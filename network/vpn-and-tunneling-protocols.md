 # VPN & Tunneling Protocols

---

# VPN (Virtual Private Network)

A VPN allows secure communication between two or more private networks over a public network (WAN such as the Internet).

It works by creating an encrypted tunnel between the client and the VPN server.

---

## VPN Components

### VPN Client
- Software or hardware used by the user
- Connects the device to the VPN server

---

### VPN Server
- Receives VPN connections
- Authenticates users
- Routes traffic securely between networks

---

## VPN Function
- Connects multiple LANs over WAN
- Creates an encrypted tunnel
- Protects data from interception

---

# VPN Protocols

## PPTP (Point-to-Point Tunneling Protocol)

- One of the oldest VPN protocols
- Easy to set up
- Fast but not very secure by modern standards

---

## L2TP (Layer 2 Tunneling Protocol)

- Used for creating VPN tunnels at Layer 2
- Often combined with IPsec for encryption (L2TP/IPsec)
- More secure than PPTP

---

## 🧠 Summary

VPN technology enables secure communication over public networks by creating encrypted tunnels. VPN clients connect to VPN servers using tunneling protocols like PPTP and L2TP to ensure privacy and secure data transmission.
