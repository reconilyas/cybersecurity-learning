# Ethernet & Wireless Networking

---

## 🌐 Ethernet Network Basics

Ethernet is a wired networking technology used for communication between devices.

### Common components:
- Ethernet cables  
- Router  
- Switch  
- Hub  

---

## 📡 Wireless LAN (WLAN)

WLAN allows devices to connect to a network wirelessly using radio waves.

---

## 🏗️ Wireless Network Types

### 🔹 BSS (Basic Service Set)
- A single access point and connected devices

### 🔹 ESS (Extended Service Set)
- Multiple access points connected together to form one larger network

### 🔹 IBSS (Independent Basic Service Set)
- Also called ad-hoc mode  
- Devices connect directly without an access point

---

## 📶 Wireless Components

### 🔹 Access Point (AP)
- Device that allows wireless devices to connect to a network

### 🔹 STA (Station)
- Any device connected to a wireless network (laptop, phone, etc.)

---

## 📡 Wireless Antennas

Antennas transmit and receive electromagnetic waves using radio frequencies.

### Gain:
- Refers to signal strength and directionality

---

## 📢 Beacon Frame
- A signal sent by an access point  
- Contains SSID (network name)  
- Helps devices discover and connect to WiFi networks  

---

## 📡 Antenna Types

- Directional → focused signal in one direction  
- Omnidirectional → signal spreads in all directions  

---

## ⚡ Wireless Transmission Types

- Infrared → requires direct line of sight (LOS)  
- Bluetooth → short range (~10–20 meters)  
- Microwave → long-distance communication between stations  
- Radio → depends on antenna strength and frequency  

---

## ⏱️ Latency
Latency is the delay between sending data and receiving it at the destination.

---

## 📡 IEEE 802.11 Standards

| Standard | Frequency | Range |
|----------|----------|-------|
| 802.11 (1997) | 2.4 GHz | ~20 m |
| 802.11a | 5 GHz | ~20 m |
| 802.11b | 2.4 GHz | ~100 m |
| 802.11g | 2.4 GHz | ~100 m |
| 802.11n | 2.4 / 5 GHz | ~70 m |

### Notes:
- Higher frequency = faster speed but shorter range  
- Lower frequency = longer range but lower speed  

---

## 📡 802.11 Modes

- Infrastructure Mode → uses AP (BSS / ESS)  
- Ad-hoc Mode → direct device-to-device (IBSS)  

---

## 🧠 Summary
Wireless networking uses radio signals to connect devices. Understanding WiFi standards, antenna types, and network structures (BSS, ESS, IBSS) is essential for networking and cybersecurity.
